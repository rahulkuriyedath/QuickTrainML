QuickTrainML

A GUI based tool for rapid ML training

Overview

QuickTrainML allows you to quickly and easily train machine learning models on your own data WITHOUT having to write a single line of code. Simply point it to the data file (like a CSV file), choose the columns to use as features, the target column that you would like to predict and the machine learning algorithm you would like to use, and voila, you have a trained model that you can run inference on!

Key features:

  • Choose from a variety of classification and regression algorithms for training including linear models (Linear and Logistic Regression), tree-based models(Random Forest), and Support Vector Machines
  • Get AI powered recommendations that provide concrete steps you can take to improve model performance. Scroll to the bottom to see it in action!
  • Perform exploratory data analysis using just the GUI, allowing you to visualize data distributions, feature interactions, etc.
  • Set custom hyper-parameters while training, with support for hyper parameter optimization
  • Visualize and compare the performance of various trained models to decide which one works best for your requirement.

Exploratory Data Analysis

Before training your machine learning model, you need to have a clear picture of what your dataset looks like. Use QuickTrainML's EDA Report builder to explore your dataset, learn more about what the data looks like and understand which features could be useful to train your machine learning model.
